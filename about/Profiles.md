### Профили запуска сервиса

* ```local-embedded``` Запускает приложение локально без внешней инфраструктуры. Идеально для отладки из idea
* ```local``` Запускает приложение локально с использованием внешней инфраструктуры в докере. Для idea тоже подходит
* ```dockerized``` Запускает приложение и инфраструктуру в docker-compose
* ```k8s``` Запускает приложение в k8s на релизном стенде
